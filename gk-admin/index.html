Cómo funcionan los tags

Cada tag agrupa entradas en el mapa del archivo. Entradas con el mismo tag forman un cluster y aparecen juntas.

Una entrada con dos tags aparece entre los dos clusters, conectándolos visualmente. Más de dos tags no tiene efecto adicional.

Usa siempre los mismos tags para el mismo tema. Si escribes un tag nuevo, se creará un cluster nuevo en el mapa.
